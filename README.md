# logica-programacao-python
# Logica de Programacao e Automacao com Python

Este repositorio centraliza exercicios e projetos desenvolvidos para o estudo de logica de programacao, estruturas de dados e automacao de calculos utilizando Python.

## Projetos em Destaque

### Calculadora de Periodizacao (1RM)
Ferramenta para automacao do planejamento de cargas em treinos de musculacao. O sistema processa a repeticao maxima (1RM) do usuario para gerar uma planilha de intensidades progressivas ao longo de um ciclo de 4 semanas.

**Destaques Tecnicos:**
* **Tratamento de Excecoes:** Implementacao de blocos try/except para garantir a robustez do software, evitando crashes por entradas de dados invalidas.
* **Validacao de Dados:** Uso de loops de repeticao (while/for) para assegurar a integridade das informacoes coletadas.
* **Normalizacao de Strings:** Tratamento de inputs com metodos strip e lower, alem de suporte a decimais com ponto ou virgula.

## Organizacao do Repositorio

* **/Projetos**: Scripts complexos que integram diversos conceitos de programacao e tratamentos de erro.
* **/Exercicios_Basicos**: Scripts focados em fundamentos como operacoes aritmeticas, manipulacao de strings e tipos de dados.

## Tecnologias e Versoes
* **Linguagem:** Python 3.14
* **Bibliotecas Standard:** math (Calculos trigonometricos e truncamento), random (Algoritmos de sorteio e embaralhamento).
* **Bibliotecas Externas:** pygame (Processamento de audio).
